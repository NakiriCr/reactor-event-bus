## 简介
为支持更高版本的`Reactor`，基于`reactor-bus`的设计，移植和重实现事件总线。

## 特性
* 适配 `Reactor3.X`
* 事件非阻塞式消费
* 多种主键匹配模式
* 发布订阅模型
* 请求应答模式
* ...
